## 1. 서비스 개요

| 항목 | 내용 |
|------|------|
| 서비스명 | CampusCraft (캠퍼스크래프트) |
| 도메인 | campuscraft.xyz |
| 슬로건 | 새 학기엔 CC하자! |
| 서비스 유형 | 대학생 전용 마인크래프트 서버 |
| 핵심 가치 | 대학교 소속감 기반의 경쟁/친목형 멀티플레이 경험 |

### 1.1 서비스 요약

CampusCraft는 대학생만 참여할 수 있는 **대학교 대항전 마인크래프트 서버**다. 대학 이메일(.ac.kr / .edu)로 본인 인증을 거쳐 서버에 입장하면, 소속 대학교 팀에 자동 배정되어 자원 수집과 영토 확장을 통해 학교 간 경쟁을 펼친다.

### 1.2 문제 정의

- 기존 마인크래프트 서버는 익명성이 높아 커뮤니티 소속감이 부족하다
- 대학생 간 교류 채널이 제한적이며, 게임을 통한 대학 간 경쟁 콘텐츠가 없다
- 서버 내 트롤링, 그리핑 등 비매너 행위에 대한 통제가 어렵다

### 1.3 해결 방안

- 대학 이메일 인증으로 실명성 확보 및 진입 장벽 형성
- 대학교 단위 팀 자동 배정으로 소속감 기반 경쟁 유도
- 인증된 사용자만 접속 가능한 화이트리스트 시스템으로 서버 품질 유지

## 2. 타겟 사용자

### 2.1 주요 타겟

| 구분 | 설명 |
|------|------|
| 인구통계 | 한국 대학교 재학생 / 휴학생 |
| 연령대 | 18~26세 |
| 관심사 | 마인크래프트, 대학 커뮤니티, 경쟁형 콘텐츠 |
| 필수 조건 | 대학교 이메일(.ac.kr / .edu) 보유, 마인크래프트 Java Edition 보유 |

### 2.2 사용자 페르소나

**페르소나 A — 새내기 게이머**
- 대학 입학 후 친구를 사귀고 싶은 1학년
- 마인크래프트 경험 있음, 같은 학교 사람들과 플레이하고 싶음
- 기대: 학교 사람들과 자연스럽게 팀플레이

**페르소나 B — 경쟁 지향 유저**
- 서버 경쟁 콘텐츠를 즐기는 마인크래프트 헤비 유저
- 대학 대항전이라는 프레임이 동기부여 요소
- 기대: 학교 순위 경쟁, 영토 확장, 전략적 플레이

**페르소나 C — 커뮤니티 지향 유저**
- 학교 커뮤니티 활동에 관심이 많은 학생
- SNS, 에브리타임 등에서 활동적
- 기대: 콘텐츠 공유, 학교 자랑, 커뮤니티 소속감

## 3. 핵심 기능

### 3.1 대학 이메일 인증 시스템

```
[사전등록] → [인증 메일 발송] → [이메일 인증] → [화이트리스트 등록] → [서버 접속]
```

- 대학 이메일 도메인(.ac.kr / .edu) 검증
- 24시간 유효 인증 토큰 발급
- 인증 완료 시 마인크래프트 서버 화이트리스트 자동 등록
- 마인크래프트 스타일 HTML 이메일 템플릿

### 3.2 대학교 팀 자동 배정

- 이메일 도메인 기반 대학교 자동 감지
- 서버 입장 시 소속 대학교 팀에 자동 배정
- 대학교 고유 색상의 갑옷(과잠) 자동 지급
- 팀 전용 채팅 (`/tc` 명령어)

### 3.3 화이트리스트 시스템

- Supabase DB와 5분 주기 동기화
- 미등록/미인증 사용자 자동 킥 (하나의 메일로 여러 유저 접속 방지)
- 관리자 수동 동기화 명령어 (`/cc sync`)

### 3.4 학과 칭호 시스템

- 서버 최초 접속 시 학과(전공) 선택 또는 명령어(`/department`)로 설정
- **플레이어 닉네임 위에 학과 칭호 표시** (머리 위 네임태그)
- 채팅 포맷에도 학과 반영: `[컴퓨터과학과] sq45: 안녕하세요!`
- **on/off 토글 지원** (`/department toggle`) — 익명성을 원하는 유저는 칭호 숨김 가능
- 학과 정보 공개로 인한 내적 친밀감 형성 효과 기대
- TAB 플러그인 연동으로 탭 리스트에도 학과 표시

### 3.5 학년 레벨링 시스템

- 서버 최초 접속 시 **1학년** 칭호 자동 부여
- 경험치(활동량) 누적에 따라 **2학년 → 3학년 → 4학년** 승급
- 학년별 해금 혜택:

| 학년 | 필요 경험치 | 해금 혜택 |
|------|-----------|----------|
| 1학년 | 0 (기본) | 기본 도구 사용 가능 |
| 2학년 | 1,000 XP | 철 도구 이상 사용 가능, 추가 홈 설정 |
| 3학년 | 5,000 XP | 다이아 도구 사용 가능, 팀 채팅 이모지 |
| 4학년 | 15,000 XP | 네더라이트 도구 사용 가능, 특수 칭호 |

- 학년 칭호도 닉네임 위에 표시 가능 (학과 칭호와 함께 또는 선택 표시)
- 경험치 획득: 자원 채굴, 건축, 퀘스트 완료, 접속 시간 등 다양한 활동

### 3.6 투표 시스템

- 서버 콘텐츠/이벤트 관련 유저 의견 수렴용
- **방식 1:** 크리에이터(유튜버/스트리머) 협업 시 크리에이터 채널 투표 활용
- **방식 2:** 인스타그램에 링크를 달아 **익명 투표** (외부 투표 도구 연동)
- 투표 주제 예시: 이벤트 종류 결정, 맵 테마 선택, 신규 기능 우선순위 등

### 3.7 닉네임 변경

- 이메일 인증 기반 닉네임 변경 지원
- 1시간 유효 변경 토큰 발급
- 닉네임 규칙: 3~16자, 영문/숫자/밑줄만 허용

### 3.8 랜딩 페이지

- 서비스 소개 및 특징 안내
- 사전등록 폼 (이메일 + 마인크래프트 닉네임)
- 참여 대학교 쇼케이스
- FAQ 섹션
- 반응형 디자인 (모바일 최적화)

## 6. 사용자 플로우

### 6.1 사전등록 플로우

```
1. 랜딩 페이지 방문
2. 대학 이메일 + 마인크래프트 닉네임 입력
3. 이메일 도메인 검증 → 대학교 자동 감지
4. 인증 메일 발송 (24시간 유효)
5. 이메일 내 인증 버튼 클릭
6. 인증 완료 페이지 표시
```

### 6.2 서버 접속 플로우

```
1. 마인크래프트 Java Edition 실행
2. 멀티플레이 → 서버 주소 입력 (campuscraft.xyz)
3. 서버 진입 시 닉네임 기반 인증 확인
   ├─ 미등록 → 킥 (등록 안내 메시지)
   ├─ 미인증 → 킥 (인증 안내 메시지)
   └─ 인증 완료 → 입장 허용
4. 소속 대학교 팀 자동 배정
5. 대학교 색상 과잠(갑옷) 자동 지급
6. 플레이 시작
```

### 6.3 닉네임 변경 플로우

```
1. /change-nickname 페이지 접속
2. 등록된 이메일 입력
3. 닉네임 변경 인증 메일 발송 (1시간 유효)
4. 이메일 내 링크 클릭
5. 현재 닉네임 확인 → 새 닉네임 입력
6. 변경 완료
```
